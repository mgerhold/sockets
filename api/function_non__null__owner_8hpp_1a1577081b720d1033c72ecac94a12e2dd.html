template<typename T, typename ...Args>
NonNullOwner<T> c2k::make_non_null_owner(Args&&... args)

Creates a NonNullOwner object.

This function creates and returns a NonNullOwner object, which is used to manage ownership of a dynamically allocated object that cannot be nullptr.

See also

NonNullOwner

Template Parameters:

  • T – The type of object to be owned.

  • Args – The types of arguments to be passed to the object’s constructor.

Parameters:

args – The arguments to be forwarded to the object’s constructor.

Returns:

A NonNullOwner object that owns the newly created object of type T.
