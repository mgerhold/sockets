template<typename T>
class BidirectionalChannel

A bidirectional communication channel that allows sending and receiving data from both ends.

Public Functions

inline void send(T value)

Sends a value using the m_sender object.

Parameters:

value – The value to be sent.

Throws:

ChannelError – If the channel has already been closed or if try_send() is called on a moved-from channel.

inline bool try_send(T value)

Attempts to send a value through the channel without blocking.

Parameters:

value – The value to be sent through the channel.

Throws:

ChannelError – If the channel has already been closed or if try_send() is called on a moved-from channel.

Returns:

True if the value was successfully sent, false otherwise.

inline T receive()

Template Parameters:

T – The type of the value to receive.

ChannelError – If the channel has already been closed or if try_send() is called on a moved-from channel.

Returns:

The received value.

inline std::optional<T> try_receive()

Attempts to receive a value from the channel without blocking.

Template Parameters:

T – The type of the value to receive.

Throws:

ChannelError – Thrown when the channel has already been moved from or closed.

Returns:

An optional containing the received value if available, or std::nullopt if no value is available.
