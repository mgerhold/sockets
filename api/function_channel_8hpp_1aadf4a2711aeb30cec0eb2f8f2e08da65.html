template<typename T>
std::pair<BidirectionalChannel<T>, BidirectionalChannel<T>> c2k::create_bidirectional_channel_pair()

Creates a pair of bidirectional channels.

This function creates a pair of bidirectional channels. It returns an std::pair containing two instances of BidirectionalChannel<T>, where T is the type of the channel values.

Template Parameters:

T – The type of the channel values.

Returns:

A std::pair containing two instances of BidirectionalChannel<T>.
